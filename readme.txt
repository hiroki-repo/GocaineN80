Welcome to Gocaine N80
---日本語---
このプログラムを使うためにはPC-8001のスイッチオンベーシックファームウエア(n80basic.rom)とフォント(font.rom)が必要ですがこちらは各自でご用意ください。
また、PC-8001のフォントとファームウエアは先ほどかっこの中に書いたファイル名でGocaineN80.exeがあるフォルダーに配置してください。
さらにこのプログラムにはCMTを読み込む機能がありますがこちらはF9を実行中に押してファイルダイアログでカセットテープデータを選んでください。
このプログラムは以下のライセンスのコードを搭載しています。
	--------------------------------------------------------------------------------
	OggVorvis形式のファイルデコードに以下のライブラリを使用しています
	libvorbis : Copyright (c) 2002-2004 Xiph.org Foundation
	--------------------------------------------------------------------------------
